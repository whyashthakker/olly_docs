This guide will walk you through the process of obtaining an API key for Olly's Language Model.

![Olly Setup](/images/installation/olly_api.png)

## Steps to Obtain Your API Key

1. **Access the Olly Dashboard**
   - Visit the [Olly dashboard](https://www.olly.social/dashboard).

2. **Log In or Sign Up**
   - If you have an account, log in.
   - If you're new, sign up for an Olly account.

3. **Generate API Key**
   - Click on the **"Generate Olly API key"** button.

4. **Receive Your API Key**
   - Once created, you'll be provided with your new API key.
   - Make sure to copy and store this key securely, as it may not be shown again.

## Ensuring API Usage

5. **Check Your Account Balance**
   - To use the API, your account needs sufficient credits.
   - Check your balance by:
     - Going to your profile > Credits
     - Or clicking [here](https://www.olly.social/credits)

6. **Top Up If Necessary**
   - If your account balance is zero, you'll need to add funds to continue using the API.

## Important Notes

- Keep your API key confidential and secure.
- Regularly rotate your API keys for enhanced security.
- Monitor your usage to avoid unexpected charges.
