# How to Get Your Straico LLM API Key

This guide will walk you through the process of obtaining an API key for Straico's Language Model.

![Straico API Key Setup](/images/installation/straico.png)

## Steps to Obtain Your API Key

1. **Access the User Settings Page**
   - Visit the [Straico User Settings page](https://platform.straico.com/user-settings).

2. **Log In or Sign Up**
   - If you have an account, log in.
   - If you're new, sign up for a Straico account.

3. **Navigate to User Settings**
   - Go to **My Profile > User Settings**
   - Or click [here](https://platform.straico.com/user-settings) to access directly.

4. **Locate Your API Key**
   - Under the **"Connect with Straico API"** section, you will find your API key.

5. **Identify Your API Key**
   - Your API key should start with "yb-..." followed by alphanumeric characters.
   - Make sure to copy and store this key securely.

## Ensuring API Usage

6. **Check Your Account Balance**
   - To use the API, your account needs sufficient credits.
   - Check your balance by:
     - Visiting "Billing and Plans"
     - Or clicking [here](https://platform.straico.com/my-profile/pricing)

7. **Top Up If Necessary**
   - If your account balance is zero, you'll need to add funds to continue using the API.

## Important Notes

- Keep your API key confidential and secure.
- Regularly rotate your API keys for enhanced security.
- Monitor your usage to avoid unexpected charges.
