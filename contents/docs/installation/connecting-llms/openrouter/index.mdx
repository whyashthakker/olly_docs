This guide will walk you through the process of obtaining an API key for Open Router's Language Models.

![Open Router API Key Setup](/images/installation/openrouter.png)

## Steps to Obtain Your API Key

1. **Access the API Keys Page**
   - Visit the [Open Router Settings Keys page](https://openrouter.ai/settings/keys).

2. **Log In or Sign Up**
   - If you have an account, log in.
   - If you're new, sign up for an Open Router account.

3. **Create a New Key**
   - Click on the **"Create key"** button.

4. **Set Up Your Key**
   - Fill in the required details for your new key.
   - Confirm to create the key.

5. **Receive Your API Key**
   - Once created, you'll be provided with your new API key.
   - The key should start with "sk-or-..." followed by alphanumeric characters.
   - **Important:** Copy or write down your key immediately. You will not be able to see it again.

## Ensuring API Usage

6. **Check Your Account Balance**
   - To use the API, your account needs sufficient credits.
   - Check your balance by:
     - Visiting the credits page
     - Or clicking [here](https://openrouter.ai/credits)

7. **Top Up If Necessary**
   - If your account balance is zero, you'll need to add funds to continue using the API.

## Important Notes

- Keep your API key confidential and secure.
- Regularly rotate your API keys for enhanced security.
- Monitor your usage to avoid unexpected charges.
- Remember that you cannot retrieve your full API key after initial creation, so store it safely.
