This guide will walk you through the process of obtaining an API key for Groq's Language Model.

![Groq Setup](/images/installation/groq.png)

## Steps to Obtain Your API Key

1. **Access the API Keys Page**
   - Visit the [Groq Console Keys page](https://console.groq.com/keys).

2. **Log In or Sign Up**
   - If you have an account, log in.
   - If you're new, sign up for a Groq account.

3. **Create a New Key**
   - Click on the **"Create API key"** button.

4. **Set Up Your Key**
   - Fill in the required details for your new key.
   - Confirm to create the key.

5. **Receive Your API Key**
   - Once created, you'll be provided with your new API key.
   - The key should start with "gsk-..." followed by alphanumeric characters.
   - Make sure to copy and store this key securely, as it may not be shown again.

## Ensuring API Usage

6. **Check Your Account Balance**
   - To use the API, your account needs sufficient credits.
   - Check your balance by:
     - Going to Settings > Billing
     - Or clicking [here](https://console.groq.com/settings/billing)

7. **Top Up If Necessary**
   - If your account balance is zero, you'll need to add funds to continue using the API.

## Important Notes

- Keep your API key confidential and secure.
- Regularly rotate your API keys for enhanced security.
- Monitor your usage to avoid unexpected charges.
