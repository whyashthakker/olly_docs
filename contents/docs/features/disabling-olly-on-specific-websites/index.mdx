## What Is This Feature?
We've added a powerful new feature that lets you manage Olly's presence across different websites. Now, you can decide where Olly is active and where it stays hidden, giving you complete flexibility! This feature allows you to maintain a customized browsing experience by selectively enabling or disabling Olly on specific websites based on your needs.

## Feature Benefits
* **Customized Experience**: Control exactly where Olly appears, tailoring your browsing to your specific needs
* **Clutter-Free Workspace**: Keep your interface clean by disabling Olly on sites where you don't need it
* **Improved Focus**: Maintain a distraction-free environment on work-related or professional websites
* **Flexible Management**: Easily add or remove websites from your disabled list at any time
* **Multi-Platform Control**: Manage Olly's presence across all the platforms you use
* **Enhanced Performance**: Reduce resource usage by limiting Olly to only the sites where you need it

## How to Manage Olly's Presence

### Step 1: Open the Olly Chrome Extension
Click on the Olly icon in your browser toolbar to access the extension.

**â†’ [Learn how to install the Chrome extension here](/https://docs.olly.social/docs/installation/adding-to-chrome)**

![Open Olly Extension](/images/features/disable-olly/disabling-olly-on-specific-websites-step1.png)

### Step 2: Go to "Olly Configuration"
Navigate to the Olly Configuration inside the extension settings section where you can customize your Olly experience.

### Step 3: "Disable Olly On Specific Website "
Find this option in the settings to control where Olly appears. This section will display a list of websites where you can manage Olly's presence.

![Disable Olly On Section](/images/features/disable-olly/disabling-olly-on-specific-websites-step3.png)

### Step 4: Add Website URLs
Simply enter the URLs of the websites where you want to manage Olly's presence. You can add multiple websites to create a comprehensive list of sites where Olly should remain disabled.

**URL Format Examples:**
* `linkedin.com`
* `twitter.com`
* `facebook.com`
* `instagram.com`
* `yourworksite.com`

![Add Website URLs](/images/features/disable-olly/disabling-olly-on-specific-websites-step4.png)


## Managing Your Disabled Websites List

### Adding Multiple Websites
You can add as many websites as needed to your disabled list:

1. Enter the website URL in the input field
2. Click "Add" or press Enter
3. The website will appear in your disabled list
4. Repeat for each additional website

### Removing Websites from the List
To re-enable Olly on a previously disabled website:

1. Locate the website in your disabled list
2. Click the remove or delete icon next to the URL
3. The website will be removed from the list
4. Olly will become active on that website again

### Editing Website URLs
If you need to modify a URL:

1. Remove the existing URL from the list
2. Add the corrected URL
3. Save your changes

## Troubleshooting

If you encounter any issues:

* **Olly still appearing on disabled sites**: Clear your browser cache and cookies, then restart Chrome
* **Unable to add websites**: Make sure your Chrome browser is up to date
* **Settings not saving**: Check that you have the latest version of the Olly extension installed
* **Extension conflicts**: Try disabling other extensions temporarily to check for conflicts
* **URLs not recognized**: Ensure you're entering the correct URL format without "https://" or "www" unless necessary
* **Changes not taking effect**: Refresh the webpage or restart your browser after making changes

### Additional Troubleshooting Steps

* Make sure your Chrome browser is up to date
* Try disabling other extensions temporarily to check for conflicts
* Clear your browser cache and cookies, then restart Chrome
* Uninstall and reinstall the Olly extension if issues persist
* Check that JavaScript is enabled in your browser settings
