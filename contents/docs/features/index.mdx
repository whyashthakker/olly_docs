## Getting Started

To maximize the benefits of Olly's features:

1. Begin with Post Generation to create your content
2. Use Viral Scores to optimize before posting
3. Enable [Comments Generator](/docs/features/comment-generator) and Replies for engagement
4. Implement Summaries for content repurposing
5. Set up Custom Actions to automate your workflow

Each feature can be used independently or as part of an integrated strategy, giving you the flexibility to build a workflow that suits your needs.
