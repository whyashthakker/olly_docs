## How to Comment in Different Languages

### Step 1: Access Configuration
1. In Olly Extension go to Configuration
![Configuration Access](/images/features/native-language/configuration.png)

### Step 2: Select Language
1. In Configuration, scroll down to find "Select Language." 
2. There, choose the language in which you want to generate comments.
![Language Selection](/images/features/native-language/language-selection.png)

### Step 3: Generate Comments
1. Navigate to any social media platform
2. Use Olly to generate comments
3. View the result in your selected language
![Comment Result](/images/features/native-language/comment-result.png)

## Post Native Language Support

### What is Post Native Language Support?
The "Post Native Language" feature automatically generates comments in the language of the original post when enabled. This functionality helps maintain linguistic consistency throughout conversations and improves engagement with international audiences.

**Example**: If someone creates a post in Japanese, enabling this feature would automatically generate responses or comments in Japanese rather than in another language like English.

## How to Enable Native Language Feature

### Step 1: Access Configuration
1. Open Olly Extension and in Extension open Olly Configuration

### Step 2: Enable the Feature
1. In Olly Configuration scroll down and look for Native language and enable the option.
![Enable Feature](/images/features/native-language/enable-feature.png)

### Step 3: Use the Feature
The Native Language feature automatically generates comments in the original language of any post. For example, if you're engaging with a Spanish post, Olly will generate comments in Spanish without requiring you to manually change the language settings each time.
![Auto Language Detection](/images/features/native-language/auto-detection.png)

## Troubleshooting

If you encounter any issues during installation:
- Make sure your Chrome browser is up to date.
- Try disabling other extensions temporarily to check for conflicts.
- Clear your browser cache and cookies, then restart Chrome.
