## How to Use Olly Pro
### Step 1: Get Your Olly API Key
1. Go to your Dashboard at `https://www.olly.social/dashboard`
2. Navigate to **"Profile"**
3. Find the **"Keys and Access"** section
4. Click on **"Generate Key"** option
5. Copy your generated Olly API key

![Generate API Key](/images/features/olly-pro/step1-generate-key.png)



### Step 2: Add API Key to Extension
1. Open your Olly extension settings
2. Navigate to the **"Add License Key"** settings section
3. Paste your generated Olly API key in the designated field
4. Save your settings

![Add API Key](/images/features/olly-pro/step2-add-key.png)


### Step 3: Select Olly Pro Model
1. In the extension settings, select **"Olly API"** as your LLM provider
2. Choose **"Olly Pro"** from the model dropdown menu
3. Confirm your selection

![Select Olly Pro Model](/images/features/olly-pro/step3-select-model.png)


### Step 4: Refresh Your Browser
1. Go to any social media platform where you want to use Olly
2. Refresh the browser window before generating comments
3. This ensures the new model settings are properly loaded


### Step 5: Generate Comments
1. Start generating comments as you normally would
2. You'll immediately notice the difference in comment quality and naturalness
3. Enjoy superior engagement with more human-like responses

![Generate Comments](/images/features/olly-pro/step5-generate.png)

## Troubleshooting
If you encounter any issues:
- **Olly Pro not appearing:** Verify you've selected "Olly API" as your LLM provider
- **API key not working:** Regenerate your key from the Dashboard and update the extension
- **Credits depleting quickly:** Remember Olly Pro uses 2 credits per comment vs 1 for Basic
- **No quality difference noticed:** Ensure you refreshed your browser after changing the model
- **Extension not responding:** Clear browser cache and restart the extension
